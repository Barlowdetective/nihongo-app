关于本应用

本应用基于《新版中日交流标准日本语》初级上册（第二版）制作，人民教育出版社出版。包含完整课文、词汇及日语发音练习功能。

发音由浏览器内置 Web Speech API 提供，需要设备安装日语语音包。
